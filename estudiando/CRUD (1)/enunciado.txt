Objetivo: Practicar los conceptos de CRUD (Crear, Leer, Actualizar, Eliminar) en Node.js utilizando memoria (un array en el código) en lugar de una 
base de datos.

Requisitos:

- Crear un proyecto de Node.js y un servidor HTTP (puede ser con Express o Node puro).

- Manejar un recurso llamado usuarios. Cada usuario debe tener al menos:

- id (numérico, autoincremental)

- nombre (string)

- email (string)

- Implementar las siguientes operaciones:

a) Listar usuarios

Ruta: GET /usuarios

Devuelve todos los usuarios almacenados en memoria.

b) Obtener un usuario por ID

Ruta: GET /usuarios/:id

Devuelve el usuario correspondiente al id proporcionado.

Si no existe, devolver un mensaje de error.

c) Crear un usuario

Ruta: POST /usuarios

Recibe nombre y email en el cuerpo de la solicitud.

Crea un nuevo usuario, asignándole un id único.

Devuelve el usuario creado.

d) Actualizar un usuario

Ruta: PUT /usuarios/:id

Permite modificar nombre y/o email de un usuario existente.

Si no existe, devolver un mensaje de error.

e) Eliminar un usuario

Ruta: DELETE /usuarios/:id

Elimina el usuario correspondiente.

Si no existe, devolver un mensaje de error.

Bonus (opcional):

Validar que no se repita el email al crear un usuario.

Implementar un filtro por nombre en la ruta GET /usuarios?nombre=....
